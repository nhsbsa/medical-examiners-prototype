Sprint 13 & 14 - Case concerns
Case concerns

Hypothesis A

We believe adding a case concern will be a useful feature for users as it will help users to add concerns to a case & to be able to produce report for the quarterly report. This will help users to complete their tasks.

What we changed:

  • Add a concern
  • Ability for users to record any concerns pertaining to a case
  • Edit facility to edit details of concern
  • Cateogary of concerns are all based on the required data for the Quarterly Report

What we plan to test:

  • Are the provided fields sufficient for users to record a concern
  • Are users able to complete their task
  • User need identified to add free text box to allow users to key in the name of the Organisation the concern relates to (this was added after the UR session)
  • Using exclamation mark to to flag cases with concerns - this was identified as a user need
  • Are users able to produce QR report with all the relevant fields asked
  • ..

What we found:

  • Recorded here : https://miro.com/app/board/o9J_kgGxF1o=/
  • Fields are sufficient and seem to go down well with users
  • Exclamation mark (for cases with case concern) - was received well

Any known issues?

  • No validation errors

Sprint 14 - Search
Search functionality

Hypothesis A

We believe adding the Search functionality will be a useful feature for users as it will help users to locate cases already in the system so users can provide updates to others, add additional information within a case & this was identified as a key user need. This will help users to confidently use the service.

What we changed:

  • Added a search link on the header
  • Search criterias added based on User need : First & Last name, Case types, NHS & hosp number
  • Search result page

What we plan to test:

  • Whether or not user able to find the icon to search
  • Are the search criterias sufficient
  • Usability level of the search form
  • ..

What we found:

  • Users found this feature to be useful

Any known issues?

  • No validation errors
